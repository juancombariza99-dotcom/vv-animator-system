# vv-animator-system

Drag & drop the vv-animator folder into your "Claude Projects" folder.
This is not an individual "skill". It's a Project folder with multiple skills inside of it.
After you have dragged and dropped the folder - restart your Claude desktop app or Claude Code.

To initialize the project folder:
**Claude Cowork:** Check your "Projects". You should see vv-animator as a project and you can click into it and chat. 

**Claude Code:**  Point your environment into the folder instance.

Start by running a prompt that says "Look through this vv-animator project to understand how it works and then we will create an animation".
